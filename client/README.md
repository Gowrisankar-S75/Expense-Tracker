# Expense Tracker

A full-stack MERN Expense Tracker application with JWT Authentication.

## Features

- User Registration
- User Login
- JWT Authentication
- Add Expense
- Edit Expense
- Delete Expense
- Expense Summary
- Income & Expense Tracking
- Expense Filtering

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB Atlas
- JWT
- Axios

## Installation

### Backend

cd server
npm install

### Frontend

cd client
npm install

### Run

npm run dev
